# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0-beta] - 2026-01-16

### Added
- Initial beta release of the `toon-parse` MCP server.
- Tool: `optimize_input_context` for converting data formats to TOON.
- Tool: `read_and_optimize_file` for token-optimized code reading.
- Resource: `protocol://mandatory-efficiency` for LLM instruction.
- Basic configuration and documentation for Cursor and Claude Desktop.
- Support for official MCP registry submission via `server.json`.
